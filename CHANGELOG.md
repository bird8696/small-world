## [0.2.0] - 2025-05-28

### 추가

- 세션 기반 뉴스 시스템 (새벽/장시작/오전장/장마감 4회)
- 뉴스 1회 호출로 8개 기업 동시 생성 (API 비용 대폭 절감)
- 뉴스 피드에 세션 배지 표시
- 헤더 뉴스 버튼에 현재 세션 이름 표시
- 자동/수동 모드 전환 구조 (useNews.js 주석으로 관리)

### 수정

- JSON +부호 양수 파싱 오류 수정 (`+8` → `8`)
- max_tokens 800 → 1500 (응답 중간 잘림 방지)
- 코드블록 마커 제거 후 JSON 추출하도록 파싱 로직 강화
- AI 모델 Sonnet → Haiku 변경 (약 20배 비용 절감)

## [0.2.0] - 2025-05-27

### 추가

- 매수/매도 패널 완전 분리
- 시장가 / 지정가 주문 유형 선택
- 10% / 25% / 50% / 100% 수량 퀵 버튼
- 매수 패널: 최대 수량 · 주문 후 잔고 표시
- 매도 패널: 보유수량 · 평균단가 · 실현손익 표시
- Supabase 로그인 / 회원가입 시스템
- 유저별 계좌 (현금 + 보유종목) DB 저장
- 유저 매매 → market_shocks → 전체 시장 가격 반영
- 헤더 유저 메뉴 드롭다운 (닉네임 · 로그아웃)
- 앱 시작 시 항상 로그인 화면부터 시작

### 수정

- 뉴스 자동 생성 20초 간격으로 안정화
- StrictMode 이중 실행 버그 수정
- 호가창 RLS 정책 분리 (insert/select/update)
